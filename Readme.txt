ANALYZING 6G NETWORK PERFORMANCE THROUGH NS3 SIMULATION 

how to run?

1. Install UBUNTU OS

2. Install NS-3 Software from "www.nsnam.org"

3. extract and put terasim module from github in the contrib folder of ns3 folder or u can downlaod form ns3 official website
{https://apps.nsnam.org/app/thz/}

4. put the main file in scratch folder of ns-3

5. open terminal at ns-3 folder 
NOte: terminal must be of ns-3 not ns-3/scratch

6. put on the command "./ns3<space>run<space>scratch/filename"
Note: filetype must be .cc

7. To run the output calculation file type in "python<space>filename.py"
